Event의 종류
 * Document Loading Event
    - 서버에서 웹 문서의 코드를 가져오거나 페이지를 스크롤 하는 등
      웹 문서를 웹 브라우저 창에 보여주는 것과 관련된 이벤트
     * abort            - 웹 문서를 완전히 로딩하기 전에 멈췄을 때 호출되는 이벤트 함수
     * error            - 웹 문서가 정확히 로딩되지 않았을 때 호출되는 이벤트 함수
  ★ * load             - 웹 문서를 모두 로딩했을 때 호출되는 이벤트 함수
     * resize           - 웹 문서 화면의 크기가 변경될 때 호출되는 이벤트 함수
     * scroll           - 화면을 스크롤 할 때 호출되는 이벤트 함수
     * unload           - 마우스포인터가 화면에서 벗어날 때 호출되는 이벤트 함수
