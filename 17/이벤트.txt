Event의 종류
 * Document Loading Event
    - 서버에서 웹 문서의 코드를 가져오거나 페이지를 스크롤 하는 등
      웹 문서를 웹 브라우저 창에 보여주는 것과 관련된 이벤트
     * abort            - 웹 문서를 완전히 로딩하기 전에 멈췄을 때
     * error            - 웹 문서가 정확히 로딩되지 않았을 때
  ★ * load             - 웹 문서를 모두 로딩했을 때
     * resize           - 웹 문서 화면의 크기가 변경될 때
     * scroll           - 화면을 스크롤 할 때
     * unload           - 마우스포인터가 화면에서 벗어날 때

 * Mouse Event
    - 마우스가 행하는 모든 동작에 발생하는 이벤트
     * click            - HTML 요소를 클릭할 때
     * dbclick          - HTML 요소를 더블클릭할 때
     * mousedown        - HTML 요소에서 마우스 버튼을 누르고 있을 때
     * mouseup          - HTML 요소에서 마우스 버튼을 누르고 떼었을 때
     * mousemove        - HTML 요소에서 마우스포인터를 움직일 때
     * mouseover        - HTML 요소 안으로 마우스포인터를 옮길 때
     * mouseout         - HTML 요소 밖으로 마우스포인터를 옮길 때
